# ControlaGastos  - Sistema de Controle de Gastos


## Descrição

O **ControleGastos** é um Sistema  de controle financeiro pessoal desenvolvido para ajudar usuários a organizarem melhor sua renda mensal e suas despesas do dia a dia.

Com ele, o usuário poderá informar sua renda mensal, cadastrar gastos como água, energia, alimentação, transporte e outras despesas, além de visualizar o total gasto e o saldo restante.

## Objetivo do Projeto

O objetivo do projeto é desenvolver uma solução simples, prática e acessível para auxiliar pessoas no controle de seus gastos mensais, promovendo uma melhor organização financeira.

## Funcionalidades

- Cadastro da renda mensal;
- Cadastro de despesas;
- Listagem das despesas cadastradas;
- Cálculo automático do total gasto;
- Exibição do saldo restante;
- Interface simples e fácil de usar.

## Tecnologias Utilizadas

- HTML5;
- CSS3;
- JavaScript;
- TypeScript
- Git e GitHub.
